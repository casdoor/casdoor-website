Casdoor支持多种语言。 通过 [Crowdin](https://crowdin.com/project/casdoor-web)，我们支持中文、法文、德文、俄文、日文和朝鲜文。

Casdoor使用官方的 Crowdin cli 来同步Crowdin 的翻译。 如果您想要添加更多语言支持，请在 [我们的社区](https://github.com/casdoor/casdoor)中提出， 如果您想帮助我们加快翻译工作，可通过 [Crowdin](https://crowdin.com/project/casdoor-web)帮助我们翻译。
